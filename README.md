# Standalone Sign Language Converter

## Overview

A wearable embedded system that converts predefined hand gestures into text using flex sensors and Arduino Nano. The system helps speech-impaired individuals communicate without requiring internet connectivity or external processing.

## Features

- Real-time gesture recognition
- Flex sensor interfacing
- LCD display output
- Portable standalone operation
- Low-cost embedded solution

## Technologies Used

- Arduino Nano
- Embedded C++
- Arduino IDE
- Flex Sensors
- LCD Module

## Workflow

1. Read flex sensor values.
2. Convert analog values into digital states.
3. Identify predefined gesture patterns.
4. Display corresponding message on LCD.

## Future Improvements

- Voice output using DFPlayer Mini
- Bluetooth communication
- Mobile application support
- AI-based gesture recognition
